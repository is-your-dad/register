# 服务条款

> [English](./TERMS_OF_SERVICE.md)

欢迎使用 **is-your.dad**(下称"我们"),这是一项面向个人的免费子域名服务。使用本服务即表示您同意以下服务条款,请仔细阅读。

**最近更新**: 2026-06-22

## 1. 定义
- **"服务"**:指 is-your.dad 子域名注册服务、本 GitHub 仓库,以及相关基础设施。
- **"用户"**、**"您"**:指使用或访问本服务的任何个人。
- **"子域名"**:指任何注册在 `is-your.dad` 之下的子域名。
- **"根子域名"**:指主要注册的子域名本身(如 `example.is-your.dad`),不包括其下的"嵌套子域名"(如 `blog.example.is-your.dad`)。
- **"内容"**:指通过子域名托管或提供的一切内容。

## 2. 接受条款
当您在本仓库提交 Pull Request 或以其他方式使用本服务时,即表示您已阅读、理解并同意受这些条款约束。如不同意,请勿使用本服务。

## 3. 资格要求
使用 is-your.dad 您必须:
1. 年满 13 周岁。
2. 是自然人,以个人名义或代表非商业团体/项目使用本服务。
3. 仅出于合法目的使用本服务。
4. 注册时提供真实准确的信息。

子域名必须注册到个人 GitHub 账号,不接受注册到 GitHub 组织账号。

## 4. 服务说明
is-your.dad 允许您在 `is-your.dad` 之下注册一个子域名。本服务按"现状"提供,不保证可用性、支持或持续性。我们保留以任何理由拒绝任何注册申请的权利。

1. 任何违法行为。
2. 损害本服务的性能或可用性。
3. 宣扬暴力或伤害。
4. 骚扰、仇恨言论或歧视。
5. 色情或露骨成人内容。
6. 假冒或虚假陈述。
7. 侵犯第三方权利。
8. 商业、营利或政治目的。
9. 发送未经请求的通讯。

11. 滥用自动化(机器人、爬虫等给服务造成过度负载)。
12. 代理或 VPN 服务。
13. 在根子域名上托管仅可登录访问的网站(管理用途的登录页除外)。

上述清单非详尽,我们可随时调整。违反本条可能立即终止服务,无需事先通知。
